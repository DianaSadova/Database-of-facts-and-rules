# Программа для обработки базы данных фактов и правил на C++

## Описание
Этот код реализует систему работы с базой данных фактов и правил, которая позволяет добавлять, удалять и проверять факты, а также выводить новые факты на основе логических связей. Основная цель программы — создание базы знаний с возможностью проверки правил и генерации выводов.

## Основные компоненты кода
### 1. **Класс `DB` (База данных):**
- Хранит факты в отсортированном массиве строк.
- Методы:
  - **`add(string fact)`** — добавляет новый факт в базу данных, сохраняя порядок сортировки.
  - **`remove(string fact)`** — удаляет указанный факт из базы данных.
  - **`contains(string fact)`** — проверяет наличие факта в базе.
  - **`find(string fact, int s, int f)`** — выполняет бинарный поиск факта в базе данных.

### 2. **Класс `Fact`:**
- Описывает структуру факта с типом и аргументами.
- Проверяет наличие факта в базе данных с помощью метода **`check()`**.

### 3. **Класс `Rule`:**
- Наследуется от класса `Fact` и добавляет проверку логических правил.
- Поддерживает логические операции **И (`&&`)** и **ИЛИ (`||`)** для проверки условий.
- Метод **`check()`** проверяет выполнение правила, основываясь на базе данных.

### 4. **Класс `Complier`:**
- Отвечает за обработку запросов к базе данных.
- Метод **`answer(string request)`** проверяет факт через базу данных.

### 5. **Функция `generateNewFacts`:**
- Генерирует новые факты, исходя из существующих логических связей.
- Пример: Если факт `older(Alice, Bob)` и `older(Bob, Charlie)` существуют, программа выводит новый факт `older(Alice, Charlie)`.

### 6. **Вспомогательные функции:**
- **`split(string line, string sep)`** — разделяет строку на подстроки по заданному разделителю.
- **`parseFact(const string& fact, string& type, string& arg1, string& arg2)`** — разбирает строку факта на отдельные компоненты: тип и аргументы.

## Пример работы программы

```
int main() {

    // Добавляем факты в базу данных
    DB::main.add("older(Alice,Bob)");
    DB::main.add("older(Bob,Charlie)");
    DB::main.add("mather(Alice,Bob)");
    DB::main.add("mather(Charlie,Den)");

    // Добавляем правила для проверки
    string args1[] = { "X", "Y" };
    Rule rule1("older", 4, args1, "older(X,Z) && older(Z,Y)");
    cout << "Rule added: " << rule1.body << endl;

    // Генерируем новые факты на основе существующих
    generateNewFacts();

    // Проверяем запрос на факт
    string args[] = { "Alice", "Charlie" };
    Fact fact("older", 2, args);
    cout << "older(Alice,Charlie) ?" << endl;
    cout << "Query result: " << (fact.check() ? "True" : "False") << endl;

    // Пример для проверки правильной генерации новых фактов 
    string args2[] = { "Bob", "Den" };
    Fact fact2("mather", 2, args2);
    cout << "mather(Bob,Den) ?" << endl;
    cout << "Query result: " << (fact2.check() ? "True" : "False") << endl;

    return 0;
}
```
